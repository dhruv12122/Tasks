# Internship Tracker

## Overview

Internship Tracker is a Django-based web application designed to manage and track internship and job applications. The application allows users to record applications, monitor hiring progress, update statuses, and maintain notes throughout the recruitment process.

The primary goal of this project is to learn Django fundamentals, database design, authentication, CRUD operations, and backend development best practices.

---

## Objectives

* Learn Django Architecture
* Understand Django ORM
* Build CRUD Applications
* Practice Authentication and Authorization
* Design Relational Databases
* Learn Django Admin Panel
